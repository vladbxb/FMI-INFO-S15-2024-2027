* ### Calculul notei

```
Notă finală (nerotunjită) = min{10, (0.7 * NES + 0.1 * NS + 0.3 * NEL (Maxim 10) >= 5)}
NES = Notă examen scris: 1 - 10 >= 5
NS = Notă seminar: 0 - 1
NEL = Notă examen laborator: 1 - 10 >= 5 (Se poate susține o singură dată)
```
